# Scarlet Studio – Framer to React Landing Page

A responsive React landing page built by recreating a Framer design as part of a Web Developer task.



## 📌 Project Overview

This project converts a Framer-based design into a fully functional React application featuring routing, reusable components, and responsive layouts. Special attention was given to matching the original design while maintaining clean, scalable code practices.

## 📝 Task Details

**Project Type:** Web Developer Assignment  
**Objective:** Convert Framer design → React application  

**Focus Areas:**
- Visual consistency with original design
- Responsive layout implementation
- Component structure and reusability
- Clean routing and navigation

## ✅ Features Delivered

- Multi-page structure (Home + subpages)
- React Router-based navigation
- Blog system with dynamic routes (`/blog/:slug`)
- Projects section with detail pages (`/projects/:id`)
- Responsive Navbar & Footer
- Hero section with animations
- Services, Pricing, and Testimonials sections
- Contact form page
- Mobile-first responsive design
